# Review Checklist

- Is review actually required for this task under the active policy and risk level?
- Did an independent reviewer perform the review?
- Did the reviewer see the relevant requirement, spec, or acceptance source?
- Did the reviewer see the relevant diff or changed files?
- Did the reviewer see verification evidence when it was available?
- Is the result clearly one of: PASS / FAIL / BLOCKED?
- If review was required, does the durable record identify Reviewer and Reference?
- If FAIL, were blocking issues fixed before completion was claimed?
- If BLOCKED, did the task remain in progress?
- Did the review stay a gate instead of turning into execution control or task reprioritization?
